# Font Attributions

## Noto Sans Regular

- **Source**: [protomaps/basemaps-assets](https://github.com/protomaps/basemaps-assets)
- **URL**: `https://protomaps.github.io/basemaps-assets/fonts/Noto%20Sans%20Regular/{range}.pbf`
- **Ranges included**: `0-255.pbf`, `256-511.pbf`
- **Licence**: [SIL Open Font License 1.1](https://scripts.sil.org/OFL)
- **Original font**: [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans) by Google

These glyphs are served locally to eliminate an external RTT dependency on `demotiles.maplibre.org`.
Only the Latin + French character ranges (0–511) are included, which cover all text labels used on this map.
